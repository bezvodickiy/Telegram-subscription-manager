# Telegram Subscription Manager Bot 🚀

An autonomous solution for managing paid Telegram channels with automated payments, CRM integration (Google Sheets), and auto-kick functionality.

## Features
- ✅ **Automated Payments:** USD support via Telegram Payments.
- ✅ **Google Sheets CRM:** Real-time user data synchronization.
- ✅ **Auto-Membership Audit:** Auto-removes users after subscription expiry.
- ✅ **Admin Dashboard:** Revenue and active member statistics.

## Tech Stack
- Python 3.10+
- Aiogram 3.x
- Google Sheets API v4
- APScheduler

## Setup
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`.
3. Create a `.env` file based on `.env.example`.
4. Add your `credentials.json` for Google Cloud.
